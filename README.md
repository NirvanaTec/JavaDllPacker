# Space - JavaDllPacker

### 介绍
它可以帮助 "SpaceLoader" 生成 SpaceLoader.wsv 文件, 来让 "SpaceLoader" 识别 ".class" 文件。

### 核心技术
计算 class 加载顺序并保存<br>为 SpaceLoader 提供 多重加密。<br>为 SpaceLoader 提供 zkm/obfuscator 等混淆器分析。

### 注意事项
它有一个 fastjson2 依赖, 请确保在运行前已经添加到项目中。<br>
java.exe -cp "JavaDllPacker.jar;SpacePacker\libs\fastjson2-2.0.60.jar" space.JavaDllPacker

### 源码介绍
JavaDllPacker<br>基于Idea2024 JDK22开发。
