# Space - JavaDllPacker

### 介绍
它可以帮助 "SpaceLoader" 生成 SpaceLoader.wsv 文件, 来让 "SpaceLoader" 识别 ".class"。

### 源码介绍
JavaDllPacker<br>
基于Idea2024 JDK22开发。
